# Simulador de Transferencia de Calor (Ley de Enfriamiento de Newton)

[cite_start]Este proyecto consiste en un simulador interactivo desarrollado para calcular la temperatura final de un objeto tras transcurrir un tiempo determinado dentro de una atmósfera a temperatura constante.

## 🚀 Características Técnicas
* [cite_start]**Fórmula Aplicada:** $T = Ts + (TO-Ts) * e^{-k*t}$ 
* [cite_start]**Procesamiento (JS):** Implementación nativa con `Math.exp()` y redondeo estricto al entero más cercano mediante `Math.round()`.
* [cite_start]**Captura de Datos:** Uso obligatorio de `document.getElementById().value` para el manejo del DOM.
* [cite_start]**Diseño UI:** Estética Tech Premium Corporativa con control riguroso del Box Model estándar en CSS
