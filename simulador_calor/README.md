# Calculador de Combinaciones Complejas (Caso Sorteo)

[cite_start]Plataforma web interactiva diseñada para calcular el total de combinaciones posibles para un sorteo de selección simultánea en dos grupos independientes.

## 🚀 Características Técnicas
* **Fórmula General:** $C(n,r) = n! [cite_start]/ (r! * (n-r)!)$ 
* [cite_start]**Procesamiento (JS):** Algoritmo propio e independiente para el cálculo del factorial ($!$) de manera iterativa, sin el uso de librerías externas.
* [cite_start]**Validaciones:** Control lógico en JavaScript para evitar desbordamientos o valores incompatibles como $r > n$.
* [cite_start]**Diseño UI:** Estética Synth Minimalista con acentos eléctricos, estructurado con HTML5 Semántico.
